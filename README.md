# 项目简介

## 核心功能
- 基于YOLOv8的行人车辆轨迹跟踪预测
- 支持图片、视频及摄像头进行检测
- 支持批量检测并查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持目标跟踪并绘制目标轨迹和预测轨迹

## 适用场景
- 行人车辆轨迹分析与预测
- 视频监控目标追踪
- 图像批量检测处理
- 实时摄像头目标检测
- 交通流量分析
- 安防监控系统

## 技术特点
- 基于YOLOv8深度学习模型
- 使用VOC2007数据集（7类：飞机、自行车、巴士、汽车、摩托车、人、火车）
- 提供完整源码和训练好的模型
- 包含3000张标注数据集
- 支持Pycharm+Anaconda或VSCode+Anaconda开发环境
- 技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9
